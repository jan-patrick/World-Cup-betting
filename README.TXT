PROJEKTZIEL:            BlueJ-Projekt und Dokumentation zur WM 2018

BENUTZERHINWEISE:       Um das Programm zu starten ist eine neue Instanz der Klasse "Main" zu erstellen. Dort können alle zur Bedienung 
                        wichtigen Methoden aufgerufen werden. Die Benutzung ist durch ein User Interface gestaltet.
                        Sollten im Betrieb Änderungen vorgenommen werden, die rückgängig gemacht werden sollen oder sind die Text-Dateien, 
                        die als Speicher dienen beschädigt oder nicht vorhanden, besteht die Möglichkeit aus dem Github Projektspeicher die 
                        Vorlage zu laden. Hierfür wird eine Internetverbindung benötigt.
